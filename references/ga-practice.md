# GA政府事务实操方法论
> last-reviewed: 2026-07-10
> GA（政府事务）是准入的战略核心。核心转变：从"辅助服务"到"战略核心"——多研究政策，不只混脸熟。

## 一、GA的定位演变

### 2015年以前：辅助支持
- 角色：公司叫干啥就干啥，市场叫干啥就干啥
- 状态："三没有、两不管"——没有价格自主管理权、没有固定投放资源、没有来自市场一线的关注
- 结果：GA部门存在感不上不下

### 2015年以后：战略核心
- 不进医保→告别公立医院市场
- 不进基药→基层及二级以上用药受严重影响
- 不进中标→意味着一片市场的放弃
- 被纳入重点监控→戴着镣铐跳舞

### 当前要求：前瞻预判
- 不只是守在一亩三分地里
- 要前瞻性地为企业市场销售提出预判
- "多研究一下政策，不要想着混个脸熟就是公共事务"

## 二、GA核心工作流

### 2.1 日常政策管理
```
政策收集 → 政策解读 → 影响分析 → 内部通报
（医保/集采/基药/DRG/挂网/定价/招标）
```

| 步骤 | 具体操作 | 交付物 |
|---|---|---|
| **政策收集** | 监控官方渠道（医保局/卫健委/NMPA/CDE/招采平台）；行业媒体；学协会公告 | 政策周报 |
| **政策解读** | 走§3三层横切规则（溯源+版本对比+5维解读）；标注对准入各簇的影响 | 政策解读报告 |
| **影响分析** | 评估对本品/竞品/治疗领域的具体影响；按紧迫度和影响度分级 | 政策影响评估报告 |
| **内部通报** | 向准入团队/市场/销售/注册通报关键政策变化；附行动建议 | 内部政策简报 |

### 2.2 政策反馈与申诉（GA专业壁垒）

> GA区别于"跑关系"的核心——能在政策制定过程中提供专业意见。

**征求意见稿阶段**：
1. 关注各部门（医保局/卫健委/药监局/CDE等）的征求意见稿发布
2. 研判政策对本品/竞品/治疗领域的影响
3. 协调企业内部资源（准入+医学+HEOR+法务）形成统一反馈意见
4. 通过正式渠道提交反馈（线上意见征集/书面反馈/座谈会）
5. 追踪反馈意见的采纳情况

**申诉阶段**：
1. 政策/目录公示后，研判是否有申诉必要和申诉空间
2. 准备申诉材料（证据+数据+理由+建议替代方案）
3. 在申诉期内提交申诉
4. 追踪申诉进程和结果
5. 申诉失败→准备替代方案（院外/商保/创新支付等）

### 2.3 政府关系维护与沟通

**拜访原则**：
- 不是"送礼混脸熟"，而是"带着政策理解去交流"
- 了解政府部门的痛点和关注点
- 提供行业视角的专业建议

**与不同部门的沟通重点**：
| 部门 | 沟通重点 | 核心话术 |
|---|---|---|
| **医保局** | 药品价值/经济性/预算影响/创新性 | "本品的ICER为X，低于1-3倍人均GDP/QALY阈值" |
| **卫健委** | 临床价值/基层适用/患者受益面 | "本品填补了XX治疗领域的空白，适宜基层使用" |
| **公共资源交易中心** | 价格合理性/供应保障/市场竞争 | "本品价格已经过全国最低价联动" |
| **药监局/CDE** | 注册进展/创新性/安全性 | （→RAskill主责，GA配合沟通） |

**与行业学会/协会联动**：
- 参与学会/协会的政策研讨会、论坛
- 联合提案/建议——汇集行业力量推动政策优化
- 通过行业共识影响政策制定

### 2.4 企业内部资源协调

GA是"协调者"——需要整合企业内多部门资源：

| 协调场景 | 需要协同的部门 | 协同内容 |
|---|---|---|
| 医保谈判准备 | 准入+HEOR+医学+法务 | ICER测算/价值档案/临床证据/合规审核 |
| 政策征求意见稿反馈 | 准入+法务+市场+注册 | 政策影响评估/法务意见/市场影响/注册可行性 |
| 集采投标策略 | 准入+供应链+财务+法务 | 报价策略/供应能力/成本测算/合规 |
| 政策倡导 | 准入+法务+市场+PR | 政策主张/法务审查/市场影响/对外传播 |

## 三、GA日常工作节奏

| 频率 | 工作内容 | 交付物 |
|---|---|---|
| **每日** | 监控政策渠道（官网/公众号/行业媒体）；处理紧急政策变化 | 政策早报（可选） |
| **每周** | 政策收集汇总+解读；内部通报；如有征求意见稿→协调反馈 | 政策周报/解读简报 |
| **每月** | 政策影响综合评估；政府拜访记录汇总；行业动态月度分析 | 月度政策分析报告 |
| **每季度** | 季度政策趋势预判；政策倡导方案更新；GA年度计划回顾 | 季度政策趋势报告 |
| **年度** | 年度政策环境扫描；GA年度策略制定；政策倡导年度计划 | 年度政策策略报告 |

## 四、GA核心能力要求

| 能力 | 要求 |
|---|---|
| **政策研究能力** | 能读懂政策原文，能做版本对比，能预判政策走向——这是GA区别于"关系型GA"的核心壁垒 |
| **专业沟通能力** | 能用专业语言与政府部门沟通（不是"拉关系"，是"交换专业意见"） |
| **跨部门协调能力** | 能协调准入/医学/法务/市场/注册等多部门资源，形成统一对外主张 |
| **前瞻预判能力** | 能从政策信号预判未来趋势，提前2-3个季度给销售和市场团队预警 |
| **持续学习能力** | 政策频繁变化，GA必须保持高频学习——"年长的学不动、年轻的吃不了苦"都是致命伤 |

## 五、GA的致命误区

| 误区 | 说明 | 正确做法 |
|---|---|---|
| **只跑关系不研究政策** | 认识多少专家/官员不是能力，专业才是 | 多研究政策原文、做版本对比、看解读分析 |
| **被动应付政策变化** | 被变化的政策牵着走 | 前瞻性预判+提前建立应对方案 |
| **以为关系能救命** | 天真地以为关键时刻会有人放一马 | 政策面前人人平等，靠证据说话 |
| **不协调内部资源** | GA单打独斗，不整合企业力量 | 协调多部门形成统一主张 |
| **不参与征求意见** | 等政策正式发布了才被动应对 | 在征求意见阶段就主动反馈，影响政策制定 |
