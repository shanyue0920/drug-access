# 各省挂网差异矩阵
> last-reviewed: 2026-07-10
> 26省挂网新规差异分析（截至2025年7月，各省陆续落地中）
> 准入Skill必须按省给出差异化建议，不给"全国通用"笼统结论

## 一、全国挂网共识框架

2024年由河北、浙江、广东、贵州、新疆五省联合形成《省级医药采购平台药品挂网规则共识》，从倡议稿到正式版。
- **"三个统一"**：统一价格形成机制、统一价差管控标准、统一监管尺度
- 26个省级单位已下发挂网新规征求意见稿/正式稿
- 各省在共识框架下结合本地实际做了针对性调整

## 二、各省挂网窗口期差异

| 申报模式 | 说明 | 典型省份 |
|---|---|---|
| **月度集中受理** | 每月固定时间窗口集中受理核验 | 福建、江西、新疆、四川、江苏等 |
| **滚动受理** | 随时申报，定期核验 | 其他多数省份 |
| **按需受理** | 特殊情况单独受理 | 短缺药/创新药等特殊品种 |

**药企应对**：
- 月度集中受理省：必须卡准每月申报窗口，错过等下月
- 滚动受理省：可灵活安排，但需关注核验周期
- 新上市药品：首发价格挂网有优先通道

## 三、各省价格联动规则差异

### 3.1 联动模式

| 模式 | 说明 | 典型省份 |
|---|---|---|
| **全国最低价联动** | 挂网价不得超过全国省级平台最低挂网价 | 多数省份 |
| **区域联动** | 特定区域间价格联动 | 京津冀联动/川藏联动/上海中部多省联动 |
| **跨区域联动** | 全国最低价+区域联动叠加 | 部分省份 |

### 3.2 特殊联动规则

| 特殊品种 | 联动规则 | 典型省份 |
|---|---|---|
| **国采非主供区** | 非主供区按不高于中选价1.5倍或同品种最高中选价挂网 | 黑龙江、重庆等 |
| **短缺药品** | 排除价格风险后直接挂网 | 湖北等 |
| **创新药** | 差异化联动范围要求 | 部分省份 |

### 3.3 企业主动联动时限

| 省份 | 要求 | 处罚 |
|---|---|---|
| **贵州** | 其他省出现新低价后**3个月内**申请联动调整 | 暂停交易且**1年内不得恢复挂网** |
| 其他省 | 各省不同（3-6个月不等） | 暂停挂网/降级/信用扣分 |

**药企应对**：
- 建立全国挂网价监控台账
- 一省降价→立即评估全国联动影响→在限期内主动申请联动
- 不主动联动→被智能比价系统发现→暂停挂网+信用黑名单

## 四、各省红黄标标准差异

### 4.1 红黄标倍数标准

| 药品类型 | 黄标阈值 | 红标阈值 | 说明 |
|---|---|---|---|
| **化学药** | 最低挂网价 **1.8倍** | 最低挂网价 **3倍** | 超1.8倍亮黄标预警 |
| **生物药** | 最低挂网价 **3倍** | 最低挂网价 **5倍** | |
| **中成药** | 最低挂网价 **3倍** | 最低挂网价 **5倍** | |

### 4.2 精细化标识

| 省份 | 标识系统 | 特点 |
|---|---|---|
| **浙江** | "红黄双色四级"预警 | 按与预警基准的倍数差异精细化分级 |
| 其他省 | 红黄双色 | 通常两级（黄/红） |

### 4.3 过评药品价格上限

| 类型 | 价格上限 | 说明 |
|---|---|---|
| **过评同通用名药** | ≤参比制剂 **70%** | 多数省份要求 |
| **未过评同通用名药** | ≤参比制剂 **60%** | 多数省份要求 |

### 4.4 国采梯度降价

| 类型 | 要求 |
|---|---|
| 国采未中选药品 | 按梯度降价要求调整价格 |
| 非主非备中选药品 | ≤中选价1.5倍 或 本省最高中选价 |

## 五、价格治理工具

### 5.1 十大工具（2025年）

| 工具 | 说明 | 影响 |
|---|---|---|
| 全国监测价 | 以全国监测价为锚点 | 所有省份挂网价基准 |
| "四同药品"治理 | 同通用名/同剂型/同规格/同厂家 | 2024年开展 |
| "三同注射剂"治理 | 同通用名/同剂型/同规格注射剂 | 2025年开展 |
| 比价小程序 | 医保定点药店药品公开比价 | 各省上线 |
| 智能比价系统 | 实时追踪全国价格波动 | 自动预警，不能侥幸 |
| 全国挂网价格一览表 | 依托医保信息平台 | 全国价格透明 |
| 药品追溯码 | 全链条应用 | 流通环节可追溯 |
| 医药价格指数 | 价格指数机制 | 趋势监测 |
| 信用评价制度 | 医药价格和招采信用评价 | 失信约束 |
| 反垄断执法 | 依法惩处垄断涨价/操纵市场 | 法律震慑 |

### 5.2 管控力度层级化

```
限期整改 → 直接暂停挂网 → 强制降价 → 设置禁入期
（轻）                                            （重）
```

- 价格管控标准从简单倍数限制 → 考虑日均治疗费用/企业数量等多因素综合标准
- 针对不同类别药品设置差异化标准
- 挂网规则刚性与硬度明显加强

## 七、药企挂网操作建议

### 7.1 新上市药品挂网
1. 确认首发价格（F5自评）
2. 确认产品类型（创新药/改良新药/仿制药）→ 对应不同挂网通道
3. 按各省窗口期分批申报（月度集中省卡准窗口）
4. 首发价格挂网 → 后续价格联动管理

### 7.2 已上市药品价格维护
1. 建立全国挂网价监控台账（每月核对）
2. 一省降价→评估全国联动影响→在限期内主动申请联动
3. 监控红黄标状态→超标及时调整
4. 关注智能比价系统预警→及时响应

### 7.3 集采后挂网
1. 中选药品：按中选价/协议价挂网
2. 未中选药品：按梯度降价要求调整→≤中选价1.5倍或本省最高中选价
3. 集采品种同通用名药：以中选价为基准确定医保支付标准
