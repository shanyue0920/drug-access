# 基药目录政策与准入路径
> last-reviewed: 2026-07-10
> 基药目录走卫健委渠道（非医保局），有独立遴选机制
> 2026版时隔8年更新，9月1日施行

## 一、基药制度概述

### 1.1 基本药物制度定位
- 基本药物是**医疗机构药品配备使用的依据**
- 是临床用药**优先备什么、用什么**的重要依据
- 基药目录是保障临床用药生产供应的**导向指引**，决定优先"产什么"

### 1.2 基药身份的商业价值

| 价值维度 | 说明 |
|---|---|
| **进院优势** | 基药在药事会中有优先地位；一品双规中优先保留 |
| **基层覆盖** | 基层使用量占比78%、二级74%、三级65% |
| **放量效应** | 基药身份→更容易进院→更大处方量→放量 |
| **配送保障** | 基药须保障基层和偏远地区供应配送 |
| **考核要求** | 医疗机构对基药使用有考核要求 |

### 1.3 基药使用量占比（2026版数据）
```
基层公立医疗卫生机构：78%
二级公立医疗卫生机构：74%
三级公立医疗卫生机构：65%
全国平均：71%（占配备品种使用总量）
```

## 二、基药目录管理办法（2026年新修订）

### 2.1 修订背景
- 上一版管理办法：2015年（时隔11年修订）
- 2026年2月11日：国家卫健委等**11部门**联合印发新《国家基本药物目录管理办法》
- 六个方面修订

### 2.2 关键变化
| 变化 | 旧版(2015) | 新版(2026) |
|---|---|---|
| 调整周期 | 不明确 | **原则上不超过3年** |
| 衔接机制 | 较少 | 基药与**分级诊疗/集采/支付报销**相衔接 |
| 创新药 | 未明确 | 尝试将创新药纳入遴选范围 |
| 动态调整 | 未明确 | 定期动态调整和优化 |

### 2.3 与医保目录的联动
- 基药3年一调 vs 医保1年一调 → 节奏对齐趋势
- 医联体可能是衔接关键
- 基药在医保报销中可能有优待

### 3.2 关键亮点

| 亮点 | 说明 |
|---|---|
| **首次纳入创新药** | 4种1类国产新药（3种化药生物+1种中成药） |

| **降糖版司美格鲁肽** | 诺和泰，唯一基药GLP-1周制剂（仅限二型糖尿病，不含减重） |

### 3.3 创新药进基药的遴选标准
- 临床价值高
- 受众人群较多
- 适宜各级医疗卫生机构配备使用
- 经过专家技术咨询和评审论证严格把关

### 3.4 未来趋势
- 国家卫健委明确：将缩短创新药从上市到进入基药目录的时间
- 支持创新药发展
- 在基本药物目录定期动态调整中纳入更多创新药

## 四、基药目录准入申报

### 4.1 申报渠道
- **走卫健委渠道**（非医保局）
- 国家卫健委牵头，多部门参与

### 4.2 遴选条件
1. **临床价值高**：有充分循证证据支持（RCT/Meta/指南推荐）
2. **受众人群多**：患者基数大/临床需求广泛
3. **适宜各级配备**：基层/二级/三级均能使用
4. **安全有效**：安全性特征良好
5. **剂型适宜**：有适宜各级使用的剂型规格

### 4.3 申报流程
1. 企业准备申报材料（循证证据/临床价值/受众分析/安全性/剂型适宜性）
2. 提交至卫健委
3. 专家技术咨询
4. 评审论证
5. 目录发布

## 五、基药与其他准入维度的联动

### 5.1 基药与医保联动
| 联动维度 | 说明 |
|---|---|
| 报销优待 | 基药在医保报销中可能有更高报销比例（各省不同） |
| 目录衔接 | 基药3年一调 vs 医保1年一调，节奏对齐 |
| 医联体用药目录 | 县域紧密型医联体用药目录以基药为重点 |

### 5.2 基药与医院准入联动
| 联动维度 | 说明 |
|---|---|
| 药事会优先 | 基药在药事会评审中有优先地位 |
| 一品双规 | 基药在同通用名中优先保留 |
| 基层联动 | 基层药品联动管理机制以基药为核心 |
| 使用考核 | 医疗机构对基药使用比例有考核要求 |

### 5.3 基药与集采联动
| 联动维度 | 说明 |
|---|---|
| 品种交集 | 基药品种可能同时被集采 |
| 供应保障 | 集采中选品种须保障基层供应 |
| 价格影响 | 集采可能影响基药价格体系 |

### 5.4 基药下沉基层
- 通过基层药品联动管理，规范县域紧密型医联体用药目录
- 持续扩大基层配备使用药品种类
- 推动基层药品上下联动配备使用、集中供应配送
- 药品短缺预警处置
- 保障偏远/交通不便地区的供应配送
