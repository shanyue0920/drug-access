# 政策5维解读框架 + 版本对比表模板
> last-reviewed: 2026-07-10
> 准入Skill强制规则：涉及任何政策时必须按本框架展开

## 一、政策5维解读框架

每条政策不只是列出来，必须按以下5维框架完整展开：

```
### {政策名称}（{文号}，{发布日期}）

**1. 政策说了什么**（核心内容提炼，不超过5条）
- 要点1
- 要点2
- ...

**2. 与上一版本的区别**（版本对比表，见下方模板）
- 见版本对比表

**3. 对准入工作的影响**（按簇标注影响范围）
- 影响A簇医保准入：...
- 影响B簇基药目录：...
- 影响C簇集采挂网：...
- 影响D簇医院准入：...
- 影响E簇药经价值：...
- 影响F簇定价策略：...
- 影响G簇商保创新支付：...
- 影响H簇政策情报：...
- 影响I簇上市前准备：...

**4. 企业需要做什么**（行动建议，按时间紧迫度排序）
- 🔴 立即（1个月内）：...
- 🟡 短期（3个月内）：...
- 🟢 中期（6-12个月）：...

**5. 关键时间节点**
- 发布日期：YYYY-MM-DD
- 施行日期：YYYY-MM-DD
- 过渡期：YYYY-MM-DD 至 YYYY-MM-DD
- 下次调整窗口：YYYY年（预判）

> 🔍 来源：{官网名称} | 文号：{文号} | 原文：{URL} | 截至：{YYYY-MM}
```

## 二、版本对比表模板

### 标准模板

```
## 政策版本对比：{政策名}

| 对比维度 | 旧版（{版本号/年份}） | 新版（{版本号/年份}） | 变化类型 | 对企业影响 |
|---|---|---|---|---|
| 维度1 | 旧内容 | 新内容 | 新增/删除/修改/制度化 | 影响描述+企业需做什么 |
| 维度2 | ... | ... | ... | ... |
```

### 变化类型定义

| 类型 | 定义 | 标记 |
|---|---|---|
| **新增** | 旧版完全没有的条款/机制 | 🆕 |
| **删除** | 新版去掉的条款/机制 | ❌ |
| **修改** | 已有但内容发生了变化 | ✏️ |
| **制度化** | 从临时性/探索性变为正式制度 | 📋 |

### 企业影响标注规则
- 每条变化必须标"对企业影响"——不只是说变了什么，还要说企业需要做什么
- 影响分级：🔴重大影响（需立即行动）/ 🟡中等影响（需短期跟进）/ 🟢轻微影响（了解即可）

## 三、解读示例

### 示例1：基药目录 2026版 vs 2018版

```
### 国家基本药物目录（2026年版）

**1. 政策说了什么**
- 时隔8年更新，2026年9月1日施行，共794种药品
- 新增116种（化药生物68+中成药48），调出2种化药，无中成药调出
- 首次纳入4种1类国产创新药（含泰它西普等）
- 降糖版司美格鲁肽（诺和泰）纳入，唯一基药GLP-1周制剂
- 儿童用药新增31种适宜品种，适宜儿童品规增至905个

**2. 版本对比表**

| 对比维度 | 旧版（2018版） | 新版（2026版） | 变化类型 | 对企业影响 |
|---|---|---|---|---|
| 品种总数 | 685种 | 794种（+116/-2） | 🆕新增 | 🔴 更多品种获基药身份→进院优势；需评估自家产品是否在新增名单 |
| 创新药 | 无创新药 | 首次纳入4种1类新药 | 🆕新增 | 🔴 创新药可走基药路径进基层；需研究进基药的遴选标准 |
| 儿童用药 | 儿童品种较少 | 新增31种+品规增至905个 | 🆕新增 | 🟡 儿药企业需关注儿童适宜剂型规格的增补机会 |
| GLP-1药物 | 无 | 降糖版司美格鲁肽纳入 | 🆕新增 | 🔴 GLP-1竞品需重新评估基层市场策略 |
| 调整周期 | 不明确 | 原则上不超过3年 | 📋制度化 | 🟢 可预判下一批窗口（约2029年），提前规划申报 |
| 管理办法 | 2015版办法 | 2026年新办法（11部门联合，6方面修订） | ✏️修改 | 🟡 需研究新办法对基药与分级诊疗/集采/支付报销衔接的要求 |

**3. 对准入工作的影响**
- 影响B簇基药目录：基药身份价值提升，创新药进基药路径打通
- 影响D簇医院准入：基药在药事会中优先地位强化，基层覆盖扩大
- 影响I簇上市前准备：创新药准入路径规划需纳入基药时间线

**4. 企业需要做什么**
- 🔴 立即：检查自家产品是否在2026版新增名单；如已入选，启动基层覆盖计划
- 🟡 短期：研究创新药进基药的遴选标准，为下一批（约2029年）做准备
- 🟢 中期：关注基药目录与医保目录联动趋势，调整整体准入策略

**5. 关键时间节点**
- 发布日期：2026-07-09
- 施行日期：2026-09-01
- 下次调整窗口：约2029年（3年周期）

> 🔍 来源：国家卫健委 | 原文：https://www.nhc.gov.cn/yaozs/c100098/202607/0f4f2bdcfead449f8412453373522470.shtml | 截至：2026-07
```

### 示例2：药品价格形成机制 国办发〔2026〕9号 vs 之前

```
### 药品价格形成机制新规（国办发〔2026〕9号）

**1. 政策说了什么**
- 全链条分类施策：高水平创新药/改良新药/通用名药分别定价
- 创新药首发价格机制：企业自评制度，上市初期与高投入高风险相符的价格
- 医保支付标准引导：独家药品谈判形成，非独家竞价形成
- 挂网价格管理：首发价格挂网/协议价挂网/差比价规则
- 价格治理：智能监测+追溯码+信用评价+反垄断

**2. 版本对比表**

| 对比维度 | 旧版（分散管理） | 新版（国办发9号） | 变化类型 | 对企业影响 |
|---|---|---|---|---|
| 创新药定价 | 无明确首发机制 | 首发价格企业自评制度 | 🆕新增 | 🔴 创新药企业需建立自评能力，首发定价窗口关键 |
| 分类施策 | 较笼统 | 高水平创新药/改良新药/通用名药分类 | 🆕新增 | 🔴 需先判定产品属于哪类，对应不同定价空间 |
| 医保支付标准 | 谈判形成 | 谈判(独家)+竞价(非独家)双轨 | ✏️修改 | 🟡 非独家药品需准备竞价策略 |
| 挂网管理 | 各省不一 | 全国统一共识框架+各省落地 | 📋制度化 | 🟡 需按统一框架+各省细则双重管理 |
| 价格治理 | 人工为主 | 智能比价系统+追溯码全链条 | ✏️修改 | 🔴 不能有侥幸心理，价格异常会被自动预警 |

**3. 对准入工作的影响**
- 影响F簇定价策略：首发价格自评是新机制，需建立自评能力
- 影响A簇医保准入：非独家药品竞价机制
- 影响C簇集采挂网：统一挂网框架+智能比价
- 影响H簇政策情报：需持续跟踪各省落地细则

**4. 企业需要做什么**
- 🔴 立即：创新药企业建立首发价格自评框架；梳理全国各省挂网价是否一致
- 🟡 短期：研究分类施策的具体判定标准；非独家药品准备竞价策略
- 🟢 中期：建立价格治理应对体系（智能比价监控+信用评价维护）

**5. 关键时间节点**
- 发布日期：2026-03-30
- 施行日期：2026年（各省陆续落地）

> 🔍 来源：国家医保局 | 文号：国办发〔2026〕9号 | 原文：https://www.nhsa.gov.cn/art/2026/4/14/art_53_20215.html | 截至：2026-07
```

## 四、使用规则

1. **触发场景**：用户问"XX政策最新怎么样了/变了什么"、Skill执行任何簇涉及政策、联网发现政策有新版本
2. **强制输出**：必须同时输出5维解读 + 版本对比表，缺一不可
3. **溯源要求**：每条政策必须有原文链接+文号+发布日期+截至日期
4. **影响标注**：必须按簇标注影响范围，必须给企业行动建议
