# 药物经济学方法学
> last-reviewed: 2026-07-10
> HEOR是准入第三柱(E簇)的方法学深度支撑。药经不单列，留在准入skill内。

## 一、核心指标

### 1.1 ICER（增量成本效果比）
```
ICER = (成本_试验组 - 成本_对照组) / (效果_试验组 - 效果_对照组)
     = 增量成本 / 增量效果(QALY)
```
- ICER < WTP阈值 → 具有经济性（性价比可接受）
- ICER > WTP阈值 → 不具有经济性
- **中国WTP阈值**：尚未官方公布统一阈值，学界多用 **1-3倍人均GDP/QALY**
  - 2025年中国人均GDP约8.5万元 → WTP约8.5万-25.5万/QALY

### 1.2 QALY（质量调整生命年）
```
QALY = 生命年数 × 质量权重(utility, 0-1)
```
- 1.0 = 完全健康
- 0 = 死亡
- 负值 = 比死亡更差的状态
- 权重来源：EQ-5D量表测量 / 文献引用

### 1.3 CER（成本效果比）
```
CER = 总成本 / 效果（非增量，适用于无对照的单组评价）
```

## 二、模型方法

### 2.1 Markov模型（马尔可夫模型）——最常用
**适用场景**：慢性病、肿瘤、心血管等进展性疾病，疾病可分为有限的几个健康状态。

**构建步骤**：
1. 定义健康状态（如：健康→患病→并发症→死亡）
2. 设定状态间转移概率（每个周期从一个状态到另一个的概率）
3. 设定每个状态的成本和效用值
4. 设定模型周期长度（如终身=按预期寿命计算）
5. 计算各方案的总成本和总QALY
6. 计算ICER

**示例状态转移**（肿瘤）：
```
无进展生存(PFS) → 疾病进展(PD) → 死亡
     ↑                  |
     └─── (不可逆转) ───┘
```

### 2.2 决策树模型
**适用场景**：急性病、短期干预、事件型决策（如手术vs保守治疗）。

**特点**：按事件发生概率分支展开，适合短期、一次性决策。不适合需要长期模拟的慢性病。

### 2.3 微观模拟模型
**适用场景**：人群异质性大、需要模拟个体特征差异的复杂场景。

**特点**：模拟大量个体逐一进展，计算量大但灵活度高。当Markov模型的状态划分不足以刻画异质性时使用。

## 三、预算影响分析（BIM）

### 3.1 目的
回答"把这个药纳入医保，基金要多花多少钱？"

### 3.2 构建步骤
1. **目标人群测算**：流行病学数据 → 患者数 → 适合本品的患者数
2. **市场替代假设**：本品上市后替代哪些现有药品？替代比例多少？
3. **基金支出测算**：
   - 有本品场景：本品费用 + 剩余竞品费用 + 不良反应处理费用
   - 无本品场景：现有竞品费用 + 不良反应处理费用
4. **预算影响 = 有本品场景支出 - 无本品场景支出**
5. 通常测算3-5年

### 3.3 关键假设敏感性
- 市场渗透率假设（本品能占多少份额）
- 替代品选择假设（替代哪些药）
- 价格假设（谈判后价格 vs 当前价格）

## 四、谈判底线测算

### 4.1 量价权衡模型
```
利润 = (谈判价 - 成本) × 预估销量
```
- 谈判价越高 → 销量可能越低（医保限制使用）
- 谈判价越低 → 销量可能越高（进院更容易）
- 需要找利润最大化的价格点

### 4.2 底线价格测算
```
底线价格 = max(不亏本价格, 合理利润价格)
不亏本价格 = 单位成本（含生产+研发摊销+销售成本）
合理利润价格 = 成本 × (1 + 合理利润率)
```

### 4.3 降价弹性分析
- 降价10% → 销量增加X%？ → 利润变化？
- 降价20% → 销量增加Y%？ → 利润变化？
- 找到利润不降反升的降价区间

### 4.4 BIA价格实操公式（行业实操）
```
BIA价格 ≈ 医保基金总盘预算 ÷ 提交产品预算总和 - 平均降幅区间
```
- **BIA价格 = 下限价格 & 续约价格**
- **CEA价格 = 上限价格**（模型精准度越高→价值越高→价格越高）
- 关键权衡：
  - BIA价格太低 → 第一次谈判成功率高，但**续约会面临大幅降价冲击**
  - BIA价格太高 → 第一次可能不进
  - 需结合市场销售预期来合理平衡——"太低太高都不是最优，寻求平衡"
- 医保基金总盘：每年医保基金增量
- 提交产品预算总和：所有进入价格测算阶段产品的预算之和
- 平均降幅区间：基于历史谈判的经验降幅范围

### 4.5 ICER与临床评分的联动（行业实操）
- **有CEA药品**：评分 < N分 → 降低ICER阈值对应的GDP倍数（放宽）；评分 > N分 → 提高（收紧）
- **无CEA药品**：对照临床专家评分调整价格，评分对应ICER范围：
  - 填补临床空白+完全替代 → 1-1.5GDP/QALY
  - 部分替代 → 0.8-1.0GDP/QALY
  - 辅助/补充 → 0.6-0.8GDP/QALY
- 行业俗语："50w不谈30w不进"——定价区间快速判断

## 五、参照药选择策略

### 5.1 参照药的重要性
- 参照药直接影响ICER测算（增量成本 = 本品成本 - 参照药成本）
- 参照药影响谈判底价（医保局以参照药价格为基础测算）
- 参照药选择是谈判策略的核心

### 5.2 参照药选择原则
1. **同机制/同靶点已上市药品**：最优先
2. **同适应症标准治疗（SOC）**：指南推荐的一线方案
3. **临床最常用替代品**：处方量最大的竞品
4. **医保目录内药品**：已有医保支付标准的参照

### 5.3 参照药选择策略
- 选价格较低的参照药 → 增量成本大 → ICER高 → 不利
- 选价格较高的参照药 → 增量成本小 → ICER低 → 有利
- 但参照药选择须合理（不能故意选贵的），医保局会审查

## 六、不确定性分析

### 6.1 确定性敏感性分析（DSA / One-way SA）
- 每次只变一个参数，看ICER怎么变
- 输出：龙卷风图（Tornado Diagram），按参数影响程度排序
- 作用：找出对结果影响最大的参数

### 6.2 概率敏感性分析（PSA / Probabilistic SA）
- 所有参数同时按分布随机抽样，模拟1000-10000次
- 输出：
  - 散点图（CEAC Plane）：每次模拟一个点
  - 成本效果可接受曲线（CEAC）：不同WTP阈值下本品具有经济性的概率
- 作用：评估结果的稳健性，回答"在X万/QALY的阈值下，本品有经济性的概率是多少"

### 6.3 参数分布选择
| 参数类型 | 推荐分布 |
|---|---|
| 概率 | Beta分布 |
| 成本 | Gamma分布 / LogNormal分布 |
| 效用值 | Beta分布 |
| 相对风险(RR/HR) | LogNormal分布 |

## 七、中国药物经济学评价要点

### 7.1 2025年药物经济学评价专家共识
- 中国药学会牵头，126位专家制定
- 86个国家将PEE纳入医保准入，62国(72.1%)要求提交ICER
- 中国2021年医保谈判首次引入PEE测算

### 7.2 分领域建议
| 领域 | 建议指标 | 模型选择 |
|---|---|---|
| 肿瘤 | ICER（优先低的方案） | Markov模型（PFS→PD→死亡） |
| 慢性病(糖尿病/高血压) | CER（优先低的药物） | Markov模型（并发症进展） |
| 公共卫生/疫苗 | 成本效益比 | 决策树模型 |
| 罕见病 | ICER + 预算影响（重点） | Markov模型 |

### 7.3 医保谈判中PEE的角色
- 专家评审5维之一：有效性/安全性/**经济性**/创新性/公平性
- 医保局用PEE测算底价
- 企业需提交PEE报告作为申报材料
- PE结论影响谈判底价和支付标准

## 八、证据来源与质量

### 8.1 证据来源优先级
1. 头对头RCT（最高质量）
2. 间接比较（网状Meta）
3. 观察性研究/RWE
4. 专家意见/共识

### 8.2 调用底座
- 循证三件套：调用 `drug-base/references/evidence-levels.md`
- 临床档案：调用 `drug-base` §2（疗效/安全性/PK基础数据）
- 证据生成执行：归医学skill，准入定义需求+调用

## 九、常见陷阱

| 陷阱 | 说明 | 规避方法 |
|---|---|---|
| 参照药选择不当 | 故意选贵的参照药降低ICER | 选择临床合理的参照药（SOC/最常用替代品） |
| 模型周期太短 | 只算1-2年，低估长期效益 | 按终身或指南推荐周期建模 |
| 效用值来源不可靠 | 用厂家自己的量表而非EQ-5D | 优先使用EQ-5D测量的效用值 |
| 不确定性分析缺失 | 只给点估计不给区间 | 必须做PSA+DSA |
| 成本范围不全 | 只算药费不算不良反应处理费/监测费 | 包含直接医疗成本（药费+诊疗+不良反应+监测） |
| 市场替代假设不合理 | 假设100%替代现有药品 | 基于临床实践和处方习惯设合理替代率 |
| 说明书不确定性未处理 | 剂量/时长不确定时不加处理 | 见§十应对方法 |

## 十、说明书不确定性应对（实操）

药品说明书"使用剂量不确定"或"使用时长不确定"是定价和ICER测算的常见难题。

### 10.1 使用剂量不确定
**场景**：说明书写"根据症状适量使用""按需使用""按体重或皮肤面积给药"
**影响**：无法确定单次治疗成本→无法准确计算ICER→谈判中容易被质疑
**应对策略**：
1. **专家调研**（资深KOL/临床专家问卷）：确定实际临床中的常用剂量范围
2. **RWS疗效研究**：通过真实世界数据证明低剂量也能保证疗效
3. **提交证据包**：将专家调研+RWS结果提交给医保局，作为剂量取值的依据

### 10.2 使用时长不确定
**场景**：实际临床中用药时长显著低于临床试验数据（如维持治疗实际只用3个月而非6个月）
**影响**：低估或高估总治疗成本→ICER不准→谈判中可能被压价
**应对策略**：
1. **RCT亚组分析**：对不同用药时长亚组做疗效分析，证明短时长也有效
2. **RWS补充证据**：基于真实世界处方数据，获取实际用药时长分布
3. **基于RWE弥补不确定性**：用RWE证明实际使用时总能达到合理疗效，支撑定价

### 10.3 原则
> 企业意愿必须辅以证据铺垫——不能只说"我们觉得这个剂量就够了"，要用专家调研+RWS+RCT亚组分析形成证据链条。

## 十一、常用工具与软件

| 工具 | 用途 | 适用场景 |
|---|---|---|
| **TreeAge** | Markov模型构建、CEA分析 | 肿瘤/慢性病等需要Markov模型的场景 |
| **Excel** | BIA预算影响分析、简单成本计算 | 所有场景的基础工具 |
| **Stata** | 统计分析、参数估计、PSA/DSA | 需要统计分析的场景 |
| **R** | 统计分析、建模、可视化、PSA/DSA | 需要灵活编程和可视化的场景 |
| **SPSS/SAS** | 统计分析、数据处理 | 传统统计分析、参数估计 |

**工具选择建议**：
- 树形模型（Markov/决策树）→ TreeAge
- 预算影响分析 → Excel（成本低、易共享、适合与医保局沟通）
- 统计分析和不确定性分析 → Stata 或 R
- 数据预处理和可视化 → R
