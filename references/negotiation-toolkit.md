# 医保谈判工具包
> last-reviewed: 2026-07-10
> 医保谈判4阶段完整流程 + 申报材料 + 谈判策略 + 续约 + 参照药 + 底线测算

## 一、医保谈判完整4阶段流程

### 阶段1：前期准备与申报

#### 1.1 药品申报材料准备
| 材料类型 | 内容 | 校准要求 |
|---|---|---|
| 基本信息 | 通用名/商品名/企业/剂型/规格/批准文号 | 须与NMPA批件一致 |
| 临床疗效数据 | 关键III期研究（研究名/NCT号/期刊/年份/关键终点结果） | 须可溯源至PubMed/ClinicalTrials.gov |
| 安全性报告 | 不良反应/禁忌/黑框警示 | 须与说明书一致 |
| 国内外价格对比 | 各国售价/IRP | 标注来源+年份+口径 |
| 市场销售数据 | 国内销售额/处方量 | 标注口径+年份+数据源 |
| 经济性分析 | ICER测算/价值档案/BIA | 参照 `heor-methodology.md` |

#### 1.2 申报与形式审查
- 企业通过国家医保服务平台（fuwu.nhsa.gov.cn）提交
- 医保部门形式审查（完整性和规范性）
- 符合条件的进入专家评审

### 阶段2：专家评审

#### 2.1 综合评估（五维）
| 维度 | 评估内容 | 权重 |
|---|---|---|
| **有效性** | 疗效是否优于现有治疗？关键终点结果 | 高 |
| **安全性** | 安全性特征？不良反应可控？ | 高 |
| **经济性** | ICER是否可接受？预算影响多大？ | 高 |
| **创新性** | 机制创新？填补空白？首创新药？ | 中 |
| **公平性** | 患者可及性？公平分配？ | 中 |

#### 2.2 遴选谈判药品
- 基于评审结果筛选临床价值+基金可承受的药品
- 纳入谈判名单

### 阶段3：谈判/竞价

#### 3.1 底价测算与谈判准备
- **医保局测底价**：采用PEE方法+预算影响+参照药价格
- **企业准备**：
  - 组建谈判团队（准入+医学+HEOR+GA）
  - 制定谈判策略和底线价格
  - 准备价值陈述（临床价值/经济价值/患者价值）

#### 3.2 现场谈判与竞价
1. 企业代表介绍药品特点/优势/临床价值
2. 与医保部门就支付价格多轮协商
3. 必要时现场竞价（非独家药品）
4. 明确：支付价格/支付标准/采购量/使用范围/支付期限

### 阶段4：结果公示与发布
1. 公示与异议处理
2. 正式发布与实施（次年1月1日执行）

## 二、医保目录调整时间线（一年一调）

```
5月底-6月初   发布调整工作方案+申报指南
6月1日-10日   网上申报窗口开通
6月底-7月初   形式审查结果公示
7月-9月       专家评审、测算谈判
11月          发布新版目录
次年1月1日    新版目录执行
```

## 三、谈判策略框架

### 3.1 谈判前准备

| 准备项 | 内容 |
|---|---|
| **参照药选择** | 选临床合理的参照药（SOC/最常用替代品）→ 参照 `heor-methodology.md` §5 |
| **ICER测算** | 基于参照药测算ICER → 确认在WTP阈值内 |
| **价值档案** | 临床价值/经济价值/患者价值/社会价值四维 |
| **预算影响** | 目标人群+市场替代+基金支出影响 |
| **底线价格** | 量价权衡模型 → 不亏本价+合理利润价 |
| **降价弹性** | 不同降价幅度下的销量变化预测 |
| **谈判团队** | 准入(策略)+医学(价值)+HEOR(数据)+GA(政策) |

### 3.1a 预谈判与模拟演练

> 预谈判阶段是谈判成功的关键——补充资料、一对一沟通、模拟演练，最大化产品价值。

**步骤一：补充资料**
- 在正式谈判前，医保局可能要求企业补充额外资料
- 常见补充项：亚组分析数据、RWE证据、更新的竞品价格对比、成本数据细化
- 企业应在此时主动提交有利于己方的数据

**步骤二：一对一沟通**
- 与测算专家/评审专家进行一对一技术沟通
- 沟通重点：解释模型假设、回答技术疑问、说明数据来源合理性
- 目标：消除专家对模型和数据的疑虑，提高专家对产品价值的认可
- 注意：这是技术沟通，不是谈判——用语要专业、客观、有据

**步骤三：模拟演练**
- 企业内部组织模拟谈判演练
- 角色扮演：一方扮演医保局谈判专家（压价方）、一方扮演企业谈判团队（守价方）
- 演练关键场景：
  - 医保局提出参照药选择质疑 → 如何回应
  - 医保局质疑ICER模型假设 → 如何用数据回应
  - 医保局提出大幅降价要求 → 如何分段让步
  - 医保局提出预算影响担忧 → 如何用BIA数据回应
- 演练输出：谈判脚本、应对清单、让步阶梯表

**步骤四：确认参加谈判**
- 企业在"申报模块"下载对应的"报送材料模板"，按要求填写提交
- 确认谈判类型（谈判新增/谈判续约/竞价）

### 3.2 谈判现场策略

| 策略 | 说明 |
|---|---|
| **价值先行** | 先讲临床价值和创新性，再讲经济性 |
| **锚定高位** | 初始报价适度偏高，留降价空间 |
| **分段让步** | 分步骤降价，每次让步要有理由（不是无底线降价） |
| **量价挂钩** | 争取以量换价——更低价格换更大用量保障 |
| **适应症限制** | 必要时限制报销范围换更高价格 |
| **协议期限** | 争取更长协议期换更低价格 |

### 3.3 谈判后执行
- 支付标准执行
- 各省挂网价调整
- 进院推进
- 销量监测（与谈判承诺对比）

## 四、医保续约

### 4.1 续约类型

| 类型 | 适用情况 | 降价规则 |
|---|---|---|
| **简易续约** | 协议到期、无重大变化 | 按规则降幅（通常较小） |
| **重新谈判** | 有重大变化（新增适应症/市场变化大） | 重新走谈判流程 |

### 4.2 简易续约降价规则要点
- 基于实际销售 vs 预估销售的差异调整
- 基于基金支出 vs 预估的差异调整
- 超预估较多 → 降幅更大
- 续约规则文件：每年医保局发布《谈判药品续约规则》

### 4.3 续约准备
1. 监测实际销售数据 vs 谈判承诺
2. 预判续约降幅
3. 准备续约材料
4. 评估是否需要走重新谈判（新增适应症等）

## 五、竞价策略（非独家药品）

### 5.1 适用场景
- 2026新规：非独家药品通过**竞价**方式形成支付标准
- 多家企业的同通用名药品竞争医保支付标准

### 5.2 竞价策略

| 策略维度 | 考量 |
|---|---|
| **竞价底线** | 成本底线+合理利润 |
| **竞争格局** | 有几家竞争？谁的成本更有优势？ |
| **报价策略** | 低价抢量 vs 适度报价保利润 |
| **支付标准形成** | 竞价后形成统一支付标准 |

## 六、医保支付标准

### 6.1 支付标准形成机制

| 药品类型 | 支付标准形成方式 |
|---|---|
| 独家药品（谈判） | 谈判形成支付标准 |
| 非独家药品（竞价） | 竞价形成支付标准 |
| 通用名药上市后 | 参照通用名药价格调整 |
| 集采品种 | 以中选价为基准 |

### 6.2 支付标准与价格关系
- 企业按**不高于支付标准**的价格向医保定点医药机构供应
- 向非医保定点医药机构供应的市场价格可不受支付标准约束
- 通用名药上市后，各省份应推动参照通用名药价格调整支付标准

## 七、新药纳入医保后的效果

### 7.1 数据（截至2026年5月，来源：国家医保局）
- 2025年新增谈判药品在**31万家**定点医药机构配备
- 1-5月基金支出**26.4亿元**
- 带动药品销售**39.5亿元**
- **1225万人次**用上2025年目录新增品种
- 新药纳入医保后大部分实现销售数量和费用**双增长**

### 7.2 典型案例
- 替尔泊肽注射液：在超过**1.2万家**定点医疗机构和零售药店有销售
