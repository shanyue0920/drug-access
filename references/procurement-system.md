# 集采体系全景
> last-reviewed: 2026-07-10
> 中国药品集采4层结构：国采→全国联盟→跨省联盟→省级集采

## 一、四层集采体系总览

```
第一层：国家组织集采（国采/VBP）
  ├─ 化药国采（第1-11批）
  └─ 生物药国采（胰岛素专项接续）
      ↓
第二层：全国联盟采购（国家指导、地方牵头，2025年约20个）
  ├─ 中成药联盟（湖北牵头）
  ├─ 中药饮片联盟
  ├─ 生物药联盟（干扰素等）
  └─ 其他专业特色联盟
      ↓
第三层：跨省联盟集采（多省联合）
  ├─ 京津冀"3+N"联盟
  ├─ 广东联盟（11省）
  ├─ 四川六省二区联盟
  ├─ 河南12省联盟
  ├─ 山东五省联盟
  └─ 陕西10省联盟等
      ↓
第四层：省级集采（单省组织）
  └─ 各省自行组织，覆盖前三层未覆盖品种
```

**2025年目标**：国家和联盟组织开展的药品集采品种达到700个。

## 二、第一层：国家组织集采（国采/VBP）

### 2.1 化药国采
| 批次 | 时间 | 品种数 | 特点 |
|---|---|---|---|
| 第1批 | 2018年(4+7) | 25种 | 试点开始 |
| 第2-3批 | 2019-2020 | 扩围 | |
| 第4-7批 | 2021-2022 | | 规模化 |
| 第8-10批 | 2023-2024 | | 规则优化 |
| **第11批** | **2025年** | | **新增3条资质门槛+锚点规则+按厂牌报量** |

### 2.2 第11批关键变化
1. **品种遴选规则首次明确**：首次明确品种遴选规则与排除原因
2. **投标资质门槛提升**（新增3条）：
   - 申报企业/受委托生产企业具有**2年以上**同类型制剂生产经验
   - 申报药品在通过**GMP符合性检查**的生产线生产
   - 生产线**2年内不违反GMP要求**
3. **报量按厂牌报量**：4.6万家医疗机构报量，77%报到品牌
4. **中选规则优化**：价差计算"锚点"，不再以简单最低价作为参考
5. **集采扩围**：鼓励民营医疗机构/零售药店参加报量；"三进"（进基层/进民营/进零售药店）

### 2.3 生物药国采
- 第6批：胰岛素专项（2022年首次），2024年接续采购
- 中标后按中选价确定医保支付标准

### 2.4 国采后执行问题
- 合同履约率仅82%（2023年通报）
- 短缺药品中60%为集采品种
- 低于成本中标现象、断供问题持续存在
- 中标企业供应保障能力是关键

## 三、第二层：全国联盟采购

### 3.1 概述
- 国家医保局指导，地方牵头组织
- 2025年预计约20个全国联盟采购
- 聚焦专业特色品种：中成药/中药饮片/生物药/高值耗材

### 3.2 中成药联盟集采
- **湖北联盟**：中成药集采主力
- 中成药集采特殊难点：
  - 无法做一致性评价
  - 各家产品质量标准不统一
  - 中药材受道地产区影响，价格和质量差异大
- 规则示例（湖北联盟）："30%量分给降幅最高的中选企业，70%由医疗机构自主选择"

### 3.3 中药饮片联盟集采
- 2025年新开展
- 聚焦临床常用饮片品种

### 3.4 生物药联盟集采
- 江西干扰素联盟集采为标志
- 其余多为省级探索性采购

## 四、第三层：跨省联盟集采

### 4.1 主要跨省联盟

| 联盟名称 | 牵头省 | 参与省 | 特点 |
|---|---|---|---|
| **京津冀"3+N"联盟** | 天津 | 覆盖全国各省级医保部门 | 从药品扩展到多类品种；2025年9月公布1155个品规中选 |
| **广东联盟** | 广东 | 11省 | 双氯芬酸等药品接续采购 |
| **四川六省二区联盟** | 四川 | 川/渝/黔/滇/藏/青 | 66个化学药品带量联动采购 |
| **四川五省联盟** | 四川 | 川/黔/滇/藏/青 | 大容量注射液带量联动采购 |
| **河南12省联盟** | 河南 | 12省 | |
| **山东五省联盟** | 山东 | 5省 | |
| **陕西10省联盟** | 陕西 | 10省 | |

### 4.2 联盟规则差异
- 各联盟品种选择不同
- 中选规则不同（有的看降幅、有的看绝对价格、有的综合评分）
- 量分配规则不同
- **药企需按联盟分别研究规则，不能一套策略打天下**

## 五、第四层：省级集采

### 5.1 概述
- 各省自行组织
- 聚焦前三层未覆盖品种
- 各省规则差异更大

### 5.2 省级集采追踪要点
- 各省集采品种清单
- 各省集采时间安排
- 各省集采规则（中选标准/量分配/执行周期）

## 六、集采投标策略框架

### 6.1 纳入条件判定
1. 品种是否满足集采纳入条件？（竞争充分/临床用量大）
2. 属于哪层集采？（国采/联盟/省级）
3. 预计什么时间被纳入？

### 6.2 投标策略
| 策略维度 | 考量 |
|---|---|
| **报量** | 首年约定采购量是否合理？能否满足供应？ |
| **报价** | 成本底线在哪？能接受的最大降幅？ |
| **中标底线** | 低于什么价格就不投？亏本线在哪？ |
| **竞争格局** | 有几家企业竞争？谁的产能/成本更有优势？ |
| **供应保障** | 中标后能否保障供应？断供的后果？ |

### 6.3 未中标应对
- 渠道转移（院外/零售/民营/线上）
- 价格策略（跟随降价 vs 维持价格）
- 差异化竞争（适应症/剂型/规格差异化）

## 七、集采与医保支付标准联动

| 集采类型 | 医保支付标准 |
|---|---|
| 国采中选药品 | 以中选价为基准确定支付标准 |
| 集采非中选药品 | 须按梯度降价要求调整（≤中选价1.5倍或本省最高中选价） |
| 联盟/省级集采 | 按各联盟/省规则确定 |

## 八、按品种类型的集采路径

| 品种类型 | 集采路径 | 关键差异 |
|---|---|---|
| **化药(过评)** | 国采为主 → 联盟/省级补充 | 一致性评价是门槛；竞争充分品种优先纳入 |
| **化药(未过评)** | 联盟/省级集采 | 无一致性评价门槛；各省规则不同 |
| **中成药** | 中成药联盟(湖北)为主 | 无一致性评价；质量标准不统一；道地药材影响 |
| **中药饮片** | 中药饮片联盟(新开展) | 2025年新开展；规则探索中 |
| **生物药** | 胰岛素专项+干扰素联盟+省级探索 | 生物类似药替代；参照药选择关键 |
